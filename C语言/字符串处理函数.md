### 字符串处理函数

* 字符串复制函数： strcpy()
```c
#include <stdio.h>
#include <string.h>

int main()
{
    char name[20],nameNew[20];

    printf("请输入您的姓名：\n");
    scanf("%s", name);

    strcpy(nameNew, name);

    printf("Hello,%s\n", nameNew);

    return 0;
}

```

* 字符串拼接函数： strcat()
```c
#include <stdio.h>
#include <string.h>

int main()
{
    char name[30],address[300],info[330];

    printf("请输入您的姓名：\n");
    scanf("%s", name);

    printf("请输入您的家庭地址：\n");
    scanf("%s", address);

    strcat(info, name);
    strcat(info, ",");
    strcat(info, address);

    printf("您的个人信息是：%s\n", info);

    return 0;
}

```
