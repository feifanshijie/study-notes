### 常用的数据输入与输出函数

* 字符数据输出与输入函数： putchar()、getchar()
```c
#include <stdio.h>

int main(void)
{
	/* 字符数据输出 */
	putchar('\110'); /* 此处使用的是八进制，表示字符H */
	putchar('e');
	putchar('l');
	putchar('l');
	putchar('o');
	putchar('\n');
	
	/* 字符数据输入 */
	char sex;
	/* 
	 * 不管用户输入多少字符，该函数只会获取第一个字符。
	 * getchar()函数得到的字符可以赋值给一个字符变量或整型变量，也可以不赋给任何变量，还可以作为表达式的一部分，
	 * 例如： putchar(getchar())
	 */
	sex = getchar();
	putchar(sex);
	putchar('\n');
	
	return 0;
}
```
